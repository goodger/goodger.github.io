Menu Maker

The menumaker.py program converts an easily-maintained menu definition table into emacs-lisp code for use by easy-menu. My menus contain all my commonly-used files and directories, and items to start up shell and Python sessions. Run menumaker.py -d to see a description of the data format and installation instructions.
